## Install

Follow these steps to install the program:

1. Clone the repository:
   ```bash
   git clone https://github.com/qobulovasror/irregular-verbs-mobile-app.git
   cd irregular-verbs-mobile-app
   ```
2. Install packages:
   ```bash
   npm install
   ```
3. Run the program:
   ```bash
   npx expo start
   ```
